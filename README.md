# 萌娘百科编辑存档

该仓库为个人编辑存档内容，以及部分缓存。

Archive 中的内容遵循原来的著作权许可，AIhistory 遵循其用户协议的著作权许可，其他全为个人书写，遵循 MIT 著作权许可。

不建议在此处提交 Issue ，建议在[百科](mzh.moegirl.com.cn)上直接修改页面。

## 主题色

图片的主题色由开源软件 Aves 图片信息最底下的 10 个颜色。标记使用 `$0` ~ `$9` 如果有歧义则使用 `C$0` ~ `C$9` 。

## 特殊标记

特殊标记适用于模版和 AI 提示词。

- `$内容$`：替换变量
- `%内容%`：固定含义，见下表。

## 文本文件特殊后缀

- `.ext`：标签以及个人记录信息，使用 2 个 `\n` 来分隔是之前记录的还是编辑时记录的；
- `.wiki`：Wiki 标记语言；
- `.wikiext`：有标签和个人记录信息前缀的 Wiki 标记语言，标签和正文使用 2 个 `\n` 来分隔；
- `.wikimodel`：Wiki 标记语言模版；
- `.info`：文件对应的提交信息；
- `.infomodel`：文件对应的提交信息的模版；
- `.prompt`：给 AI 的提示词，`$` 可以替换为文件或代入的内容；
- `.promptmodel`：AI 提示词的模版。
